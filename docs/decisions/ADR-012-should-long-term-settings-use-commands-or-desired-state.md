# ADR-012 — Should long-term settings use commands or desired state?

## Status

Confirmed

## Area

Commands

## Decision

Momentary actions use commands. Persistent configuration uses desired state.

## Rationale

Separates transient control from configuration.

## Consequences and trade-offs

Simpler recovery and offline reconciliation.
