# ADR-006 — Can multiple users control one room at the same time?

## Status

In Review

## Area

Sessions

## Decision

Proposed: multiple viewers, one active User controller, with Technician and Admin takeover.

## Rationale

Avoid conflicting commands while retaining support access.

## Consequences and trade-offs

Needs visible session ownership and takeover auditing.
