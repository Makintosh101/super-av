# ADR-011 — What is the default device lifecycle?

## Status

Confirmed

## Area

Lifecycle

## Decision

Prepared → Unclaimed → Claimed → Assigned → Online → Maintenance → Suspended → Retired. Keep lifecycle intentionally simple for Phase 1.

## Rationale

Simple lifecycle reduces implementation complexity.

## Consequences and trade-offs

Future states can be added without breaking existing devices.
